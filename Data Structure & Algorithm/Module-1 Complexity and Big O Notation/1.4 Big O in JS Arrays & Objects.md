# 1.4 Big O in JS Arrays & Objects

| ---               | Objects    | Arrays                            |
| ----------------- | ---------- | --------------------------------- |
| Data Structure    | Hash Table | Array                             |
| Insertion (插入)    | $O(1)$     | push -> $O(1)$, unshift -> $O(n)$ |
| Removal (移除)      | $O(1)$     | pop -> $O(1)$, shift -> $O(n)$    |
| Searching (搜尋)    | $O(1)$     | $O(n)$                            |
| Accessing (尋找對應值) | $O(1)$     | $O(1)$                            |
