# 4.2 Array
## 4.2.1 Definition

- 陣列（array）是一種線性資料結構，其將相同 Type 的元素儲存在 **連續的記憶體空間中**。我們將元素在陣列中的位置稱為該元素的 index，利用 index 可以計算出該元素對應的儲存位置。
- 也就是 JavaScript 中的 Array 或者是 Python 中的 List。
